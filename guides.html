Speed Up Slow Wi-Fi

Restart modem/router, move router to a central spot, use 5 GHz where possible, update firmware, and run a speed test near the router.

Tip: If the speed is good near the router but bad in bedrooms, consider a mesh Wi-Fi system.

Set Up a Password Manager

Choose a reputable manager, create one strong master password, enable two-factor authentication, and import or save logins as you go.

Goal: Unique passwords everywhere, auto-fill enabled on phone and computer.

Back Up Your Computer (Windows & Mac)

Use an external drive (File History or Time Machine) and add a cloud backup for extra safety. Test restoring a file to confirm it works.

Fix Common Printer Issues

Reinstall the printer, connect via Wi-Fi using the printer control panel, ensure both devices are on the same network, and print a test page.

Make Your Smart TV Work Better

Connect via Ethernet or 5 GHz Wi-Fi, remove unused apps, sign into streaming apps, and enable automatic updates.

Free Up Space on Your Phone

Delete large videos, offload rarely used apps, clear downloads, and enable automatic photo backups (iCloud/Google Photos).
